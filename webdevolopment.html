Day 1: Introduction to Web Development & HTML Basics

-What is Web Development? Frontend vs Backend.
-HTML structure ( <!DOCTYPE html>, <html>, <head>, <body> ).
-Basic tags: headings, paragraphs, links, images.
✅ Activity: Created a simple webpage with text and an image.

Day 2: HTML Forms & Tables

-Forms: <input>, <textarea>, <button>, <select>.
-Tables: <table>, <tr>, <td>, <th>.
-Attributes: id, class, name.
✅ Activity: Designed a student registration form.

Day 3: Advanced HTML & Semantic Tags

-Semantic elements: <header>, <nav>, <section>, <article>, <footer>.
-Multimedia in HTML: <audio>, <video>.
-Hyperlinks, lists (ordered/unordered).
✅ Activity: Built a mini profile webpage using semantic structure.

Day 4: Introduction to CSS

-What is CSS? Inline, Internal, External CSS.
-Selectors: element, class, id.
-Styling text (fonts, colors, background).
✅ Activity: Styled a profile page with colors and fonts.

Day 5: CSS Box Model & Layouts

-Box model: margin, border, padding, content.
-CSS positioning: static, relative, absolute, fixed.
-Flexbox basics for responsive design.
✅ Activity: Designed a simple card layout using CSS.

Day 6: CSS Advanced Styling

-Transitions & hover effects.
-CSS Grid basics.
-Responsive design with media queries.
✅ Activity: Built a responsive portfolio homepage.

Day 7: Introduction to JavaScript

-Why JavaScript? Making websites interactive.
-Variables, data types, operators.
-DOM manipulation (document.getElementById, innerHTML).
-Event handling (onclick, onmouseover).
✅ Activity: Added a button that changes text/color on click.
